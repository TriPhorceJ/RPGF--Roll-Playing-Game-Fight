# RPGF - Roll Playing Game Fight

A fighting role playing game.
You can play at different difficulties and play two player.
```
+--------------------+
|     #@@@@#         |
|    @WWWWWM@        |
|   #WWWWM$WM        |
|   #@@$@@#$$#@#     |
|    @@@@@@@@$WM$@#  |
|     @@@@@@$WWWWMM$@|
|      @@@@MWWWWWWWWM|
|       ##WMMWWWWWWMW|
|         $$MWWWWWWMM|
|         @$MMWWWWMMW|
+--------------------+
```
To compile the program run:
```
mcs game.cs
```
To run the program run:
```
./game.exe
```
